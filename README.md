# PinBoard

PinBoard public download page and release assets.

Current version: v0.1.0

Download URL:
https://github.com/Cheney-0821/pinboard-site/releases/latest/download/PinBoardSetup.exe

## Features
Screen capture, clipboard image/text, WeChat compatibility, EXIF orientation, image rotation, global hotkeys, tray resident.

## Privacy
PinBoard runs locally. No uploads. Settings stored locally.

## Repository
Public site and release assets only.
