# Release Assets

PinBoard installers are distributed via GitHub Release.

Fixed download filename: `PinBoardSetup.exe`

Fixed latest download link:
```
https://github.com/Cheney-0821/pinboard-site/releases/latest/download/PinBoardSetup.exe
```

Release rules:
1. Each release version gets a GitHub Release
2. Release tag follows `v0.1.0` format
3. Installer uploaded as a Release asset
4. Public asset filename is always `PinBoardSetup.exe`
5. The download button on the website always points to the latest download link
