PinBoard

Turn screenshots, images, and text into always-on-top desktop reference cards. Ideal for design reference, writing, and multi-window work.

Download PinBoard View on GitHub

v0.1.0 download coming soon. The button will work once a Release is uploaded.

Screenshot to Card

Select a screen region, and a draggable, zoomable, always-on-top card appears at that spot.

Clipboard to Card

Copy an image, image file, or text, then use a hotkey to create a card instantly.

Lightweight & Persistent

System tray, single instance, customizable global hotkeys, and local settings persistence.
